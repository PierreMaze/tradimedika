# Stratégie d'Acquisition B2B

## Le défi

Toucher des professionnels de santé est **plus difficile** que le grand public :

- Ils sont occupés et sceptiques
- Ils ont besoin de confiance et de crédibilité
- Les canaux classiques (réseaux sociaux, SEO) sont moins efficaces en B2B santé

---

## 1. Canaux d'acquisition par priorité

### Canal 1 — Le bouche-à-oreille professionnel (le plus puissant)

**Principe** : Un pro qui utilise et recommande à ses pairs.

**Actions** :

- Identifier 5-10 "early adopters" (pros curieux et connectés)
- Leur donner un accès gratuit à vie en échange de feedback
- Leur demander de recommander à 3-5 collègues
- Créer un programme de parrainage (1 mois gratuit par filleul)

**Pourquoi ça marche** :

- Les pros font confiance aux recommandations de pairs
- Un pharmacien qui recommande à son réseau = acquisition organique
- Coût d'acquisition quasi nul

### Canal 2 — LinkedIn (le réseau des pros)

**Principe** : Contenu éducatif + visibilité auprès des pros de santé.

**Actions** :

- Publier 2-3 posts/semaine : fiches produits, infos réglementaires, stats
- Cibler les groupes de pharmaciens, médecins, nutritionnistes
- Format : carrousels (fiches visuelles), posts courts avec données clés
- Commenter et interagir dans les groupes pro

**Exemples de posts** :

- "Saviez-vous que le millepertuis interagit avec 40+ médicaments ?"
- "Les 5 plantes autorisées pendant la grossesse (et celles à éviter)"
- "Infographie : niveaux de preuve des 10 plantes les plus populaires"

### Canal 3 — Salons et congrès professionnels

**Événements cibles** :

- **Pharmagora Plus** (salon pharmacie, Paris) — mars chaque année
- **Congrès de médecine générale** (CMGF)
- **Journées francophones de nutrition** (JFN)
- **Salon Natexpo** (produits naturels / bio)
- **Congrès de phytothérapie** (SIPF, SFE)
- **Salon des Thermalies** (bien-être, santé naturelle)

**Actions** :

- Stand ou présence (même petit stand / table)
- Démo live de l'outil
- Distribution de flyers avec QR code → essai gratuit
- Récolte d'emails → nurturing

### Canal 4 — Partenariats professionnels

**Cibles** :

- **Ordres professionnels** : Ordre des pharmaciens, Ordre des médecins
  - Newsletter, site web, recommandation d'outils
- **Syndicats / associations** : FSPF (pharmaciens), MG France (généralistes), AFDN (diététiciens)
  - Partenariats de contenu, offres spéciales pour les adhérents
- **Formations continues** : DPC (Développement Professionnel Continu)
  - Proposer Tradimedika comme outil pédagogique
- **Facultés** : Pharmacie, médecine, nutrition
  - Accès gratuit pour étudiants → futurs clients

### Canal 5 — Content marketing / SEO pro

**Blog / articles** :

- Articles de fond sur les produits naturels (SEO long-tail)
- Guides pratiques : "Comment vérifier les interactions plantes-médicaments"
- Actualités réglementaires (EFSA, ANSES, DGCCRF)
- Newsletter mensuelle pour les inscrits

**SEO cible** :

- "interaction gingembre anticoagulant" → fiche produit
- "plantes autorisées grossesse" → liste thématique
- "base de données phytothérapie professionnels" → landing page
- "complément alimentaire interaction médicament" → outil interactions

## 2. Entonnoir d'acquisition

```
DÉCOUVERTE
  │  LinkedIn, salons, SEO, bouche-à-oreille
  │
  ▼
INTÉRÊT
  │  Visite du site → version gratuite (B2C)
  │  Voit les fiches basiques → teaser des infos pro
  │
  ▼
ESSAI
  │  Inscription pro → essai gratuit 14-30 jours
  │  Accès complet aux fiches, filtres, export PDF
  │
  ▼
CONVERSION
  │  Fin de l'essai → abonnement Pro
  │  Relance email si pas converti
  │
  ▼
RÉTENTION
  │  Usage régulier → valeur quotidienne
  │  Mises à jour des fiches → raison de rester
  │
  ▼
RECOMMANDATION
     Programme parrainage → nouveaux pros
```

## 3. Argumentaire par cible

### Pour le médecin généraliste

> "Vos patients vous posent des questions sur les plantes et compléments ?
> Tradimedika vous donne les réponses sourcées en 30 secondes.
> Vérifiez les interactions, consultez la posologie, exportez une fiche patient."

### Pour le pharmacien

> "Au comptoir, un client vous demande si le curcuma est compatible avec son traitement.
> Tradimedika vous donne la réponse avec le niveau de risque et les sources EMA.
> L'outil de référence pour le conseil officinal en produits naturels."

### Pour le nutritionniste / diététicien

> "Enrichissez vos plans alimentaires avec des produits naturels documentés.
> Filtrez par propriété, vérifiez les contre-indications, exportez pour vos patients.
> Tout est sourcé, tout est professionnel."

### Pour le kinésithérapeute

> "Vos patients cherchent des compléments anti-inflammatoires naturels ?
> Tradimedika vous permet de les orienter avec confiance et documentation.
> Sans risque, sans allégation, juste de l'information fiable."

## 5. Métriques d'acquisition à suivre

- **Visiteurs uniques** (site)
- **Taux d'inscription** (visiteur → compte gratuit)
- **Taux de conversion essai** (gratuit → essai pro)
- **Taux de conversion payant** (essai → abonnement)
- **Coût d'acquisition client (CAC)**
- **Source d'acquisition** (d'où viennent les pros ?)
- **NPS** (Net Promoter Score — recommanderaient-ils l'outil ?)
