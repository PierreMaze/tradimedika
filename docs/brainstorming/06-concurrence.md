# Concurrence et Positionnement

## Paysage concurrentiel

Le marché des bases de données de produits naturels est **fragmenté** :

- Outils pro payants et anglo-saxons
- Encyclopédies francophones gratuites mais vieillissantes
- Sites grand public financés par la pub
- Aucun outil ne combine catalogue moderne + niveau pro + freemium + français

---

## 1. Concurrents professionnels

### Natural Medicines (NatMed Pro) — Leader mondial

- **URL** : trchealthcare.com
- **Prix** : 69-182$/an
- **Contenu** : 1 400+ monographies, 300 000+ produits commerciaux
- **Points forts** : Exhaustif, interactions médicamenteuses, preuves scientifiques, MAJ quotidienne
- **Points faibles** : Anglo-saxon, cher, pas adapté au contexte français/européen
- **Menace** : Faible en France (peu connu des pros français)

### WikiPhyto — Encyclopédie collaborative francophone

- **URL** : wikiphyto.org
- **Prix** : Gratuit
- **Contenu** : 1 697 articles, 48 contributeurs
- **Points forts** : Francophone, gratuit, validé par des pros reconnus, bibliographies
- **Points faibles** : Interface vieillissante, pas d'interactions, pas d'app mobile, pas d'export
- **Menace** : Moyenne (bon contenu mais UX datée)

### VIDAL Phytothérapie — Section phyto du Vidal

- **URL** : vidal.fr/parapharmacie/phytotherapie-plantes.html
- **Prix** : Gratuit (consultation en ligne)
- **Contenu** : ~80 plantes
- **Points forts** : Marque de confiance Vidal, validé, intégré à l'écosystème médicaments
- **Points faibles** : Très limité en nombre (80), pas d'outil d'interactions plantes, pas d'export
- **Menace** : Faible (trop limité)

### Base ANSES — Plantes en compléments alimentaires

- **URL** : anses.fr (fichier Excel téléchargeable)
- **Prix** : Gratuit
- **Contenu** : 118 plantes, contre-indications, interactions
- **Points forts** : Source officielle, focus sécurité, basé sur données EMA
- **Points faibles** : Format Excel (!), pas d'interface web, pas de recherche avancée
- **Menace** : Nulle (pas un concurrent direct, mais une source de données)

### EMA HMPC — Monographies européennes

- **URL** : ema.europa.eu
- **Prix** : Gratuit
- **Contenu** : 200+ monographies communautaires
- **Points forts** : Référence réglementaire européenne, validation scientifique rigoureuse
- **Points faibles** : En anglais, interface technique, pas orienté usage clinique
- **Menace** : Nulle (source de données, pas un outil pour les pros)

### PHYDGI — Interactions plantes-médicaments

- **URL** : synapse-medicine.com (intégré)
- **Prix** : Non communiqué (B2B)
- **Contenu** : 58 plantes, 114 substances médicamenteuses
- **Points forts** : Spécialisé interactions, données structurées, innovation française
- **Points faibles** : Très limité (58 plantes), pas d'interface autonome, pas grand public
- **Menace** : Faible (niche interactions uniquement)

### Thériaque — Base médicaments française

- **URL** : theriaque.org
- **Prix** : Gratuit pour pros
- **Contenu** : Tous médicaments français (y compris phyto avec AMM)
- **Points forts** : Exhaustif médicaments, officiel, indépendant
- **Points faibles** : Focus médicaments (pas les plantes en usage traditionnel), interface datée
- **Menace** : Nulle (complément, pas concurrent)

## 2. Concurrents grand public

### Doctissimo — Section phytothérapie

- **Prix** : Gratuit (publicité)
- **Points forts** : Notoriété massive, contenu accessible
- **Points faibles** : Pubs omniprésentes, fiabilité questionnée par les pros, pas d'outil pro
- **Menace** : Faible en B2B (pas crédible pour les pros)

### Passeport Santé — Section plantes

- **Prix** : Gratuit (publicité)
- **Points forts** : Contenu éditorial de qualité, bibliographies
- **Points faibles** : Pas d'outil pro, publicités, pas d'interactions
- **Menace** : Faible en B2B

### PlantNet — Reconnaissance de plantes par photo

- **Prix** : Gratuit
- **Contenu** : 20 000+ espèces
- **Points forts** : IA, gratuit, millions d'utilisateurs
- **Points faibles** : Identification botanique uniquement, pas d'infos thérapeutiques
- **Menace** : Nulle (pas le même segment)

### Apps mobiles "Plantes Médicinales"

- **Prix** : 0-15€
- **Points forts** : Accessibles, mobiles, modernes
- **Points faibles** : Qualité scientifique variable, pas de validation pro
- **Menace** : Faible (pas crédibles pour les pros)

## 4. Positionnement de Tradimedika

### Ce qu'on est

> **La première base de données française de produits naturels,
> sourcée scientifiquement, conçue pour les professionnels de santé,
> avec une interface moderne et un modèle freemium.**

### Ce qu'on n'est PAS

- Pas un Vidal (on ne fait pas les médicaments)
- Pas un Doctissimo (on ne fait pas du contenu sensationnaliste)
- Pas un WikiPhyto (on n'est pas une encyclopédie collaborative)
- Pas un Natural Medicines (on est français et accessible)

### Différenciateurs clés

| Différenciateur                          | Vs qui                                          | Pourquoi c'est un avantage                              |
| ---------------------------------------- | ----------------------------------------------- | ------------------------------------------------------- |
| **Interface moderne**                    | WikiPhyto, ANSES, EMA                           | UX pro pensée pour l'usage quotidien                    |
| **Français + contexte réglementaire FR** | Natural Medicines                               | Adapté aux listes 148/540, allégations EFSA             |
| **Freemium**                             | Natural Medicines (payant), WikiPhyto (gratuit) | Accessible + monétisable                                |
| **Export PDF professionnel**             | Tous                                            | Aucun concurrent ne propose d'export utilisable         |
| **Filtres avancés**                      | Tous                                            | Grossesse, enfants, allergènes, propriétés, preuve      |
| **Catalogue produits courants**          | WikiPhyto (plantes rares)                       | Miel, citron, gingembre = ce que les patients demandent |
| **Statut réglementaire intégré**         | Tous sauf ANSES                                 | Liste 148, 540, EFSA directement dans la fiche          |

### L'avantage du "Blue Ocean"

Aucun concurrent ne combine :

1. Catalogue de produits naturels courants (pas que des plantes rares)
2. Niveau d'information pro (sources, interactions, posologie)
3. Interface moderne et UX soignée
4. Export PDF utilisable en consultation
5. Contexte réglementaire français intégré
6. Modèle freemium (gratuit B2C + premium B2B)

→ C'est un espace vacant sur le marché.

## 5. Risques concurrentiels

| Risque                                        | Probabilité | Impact     | Mitigation                                            |
| --------------------------------------------- | ----------- | ---------- | ----------------------------------------------------- |
| WikiPhyto se modernise                        | Faible      | Élevé      | Avancer vite, se différencier sur l'UX et l'export    |
| Vidal lance une version complète phyto        | Moyenne     | Très élevé | Être déjà établi avec une base d'utilisateurs fidèles |
| Natural Medicines lance une version française | Faible      | Élevé      | Focus réglementaire FR + prix accessible              |
| Un nouvel entrant copie le concept            | Moyenne     | Moyen      | First-mover advantage + communauté de pros            |
